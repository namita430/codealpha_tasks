# AI FAQ Chatbot

This project is a simple AI FAQ chatbot built using Python, NLP, and Flask.

Technologies Used:
- Python
- NLTK
- Scikit-learn
- TF-IDF
- Cosine Similarity
- Flask

How to run:

1 Install dependencies
pip install -r requirements.txt

2 Run chatbot
cd src
python app.py

3 Open browser
http://127.0.0.1:5000
